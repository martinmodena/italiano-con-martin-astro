# Italiano con Martin

Sito statico per l'insegnamento dell'italiano, pronto per GitHub Pages.

## Contenuti iniziali
- Home orientata a studenti e potenziali nuovi allievi.
- Banner Preply e WhatsApp.
- Favola **Il cane e l'osso** in cinque livelli: A1, A2, B1, B2 e C1.
- Vocabolario e domande di comprensione/conversazione.
- SEO tecnico: metadati, dati strutturati Schema.org, sitemap, robots.txt, Open Graph.
- Visibilità per assistenti IA: contenuto HTML leggibile e file `llms.txt`.

## Pubblicazione
Il workflow `.github/workflows/pages.yml` pubblica automaticamente il sito con GitHub Pages.

URL previsto: `https://italianoconmartin.com/`

## Regola linguistica
Il passato remoto non viene usato nei livelli A1, A2 e B1. Compare soltanto dal livello B2.
