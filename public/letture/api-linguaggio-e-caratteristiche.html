A1

Le api: caratteristiche e linguaggio - livello A1

Presente

Le api sono piccoli insetti molto importanti. Vivono in gruppo, dentro un alveare.

Le api volano sui fiori e prendono il nettare. Con il nettare fanno il miele.

Quando visitano i fiori, aiutano anche le piante.

Le api comunicano con una danza: un'ape può dire alle altre dove trovare fiori buoni.

Scheda lessico

ape, fiore, miele, alveare, danza

Scheda attiva

  1. Rispondi con una frase semplice: qual è l'idea principale?
  2. Scegli tre parole e scrivi una frase per ognuna.
A2

Le api: caratteristiche e linguaggio - livello A2

Descrizione A2

Le api sono insetti sociali: vivono in una comunità organizzata.

Alcune cercano nettare e polline, altre proteggono l'alveare, altre curano le larve.

La danza delle api è affascinante. Con movimenti precisi, un'ape informa le compagne sulla direzione e sulla distanza dei fiori.

Le api sono fondamentali per l'ambiente perché aiutano molte piante a riprodursi.

Scheda lessico

insetti sociali, nettare, polline, larve, regina

Scheda attiva

  1. Racconta il testo in quattro frasi.
  2. Scrivi una domanda da fare a un compagno.
B1

Le api: caratteristiche e linguaggio - livello B1

Relazioni e cause

Le api mostrano una forma di organizzazione sorprendente. L'alveare funziona grazie alla collaborazione di migliaia di individui.

La danza delle api è uno dei sistemi di comunicazione animale più studiati.

Attraverso direzione e durata dei movimenti, un'ape indica dove si trova una fonte di cibo.

Proteggere le api significa proteggere anche agricoltura e biodiversità.

Scheda lessico

collaborazione, fonte di cibo, comunicazione animale, biodiversità, agricoltura

Scheda attiva

  1. Spiega una causa e una conseguenza presenti nel testo.
  2. Collega il tema alla tua esperienza personale.
B2

Le api: caratteristiche e linguaggio - livello B2

Lessico specifico

Le api sono spesso associate al miele, ma il loro valore ecologico va molto oltre.

Come impollinatori, contribuiscono alla riproduzione di numerose specie vegetali.

La danza dell'addome trasmette informazioni complesse su direzione e distanza.

Pesticidi, perdita di habitat e cambiamento climatico rendono la loro vita più difficile.

Scheda lessico

impollinatori, specie vegetali, danza dell'addome, pesticidi, habitat

Scheda attiva

  1. Esprimi un'opinione usando almeno un connettivo: però, inoltre, infatti.
  2. Trova un punto interessante e spiega perché.
C1

Le api: caratteristiche e linguaggio - livello C1

Divulgazione avanzata

L'alveare rappresenta un modello affascinante di intelligenza distribuita.

Nessun'ape possiede una visione completa del sistema, eppure la colonia regola lavoro, difesa, nutrizione e riproduzione.

La danza mostra come un'informazione spaziale possa essere tradotta in movimento.

Studiare le api significa osservare insieme natura, cooperazione e vulnerabilità.

Scheda lessico

intelligenza distribuita, informazione spaziale, codice biologico, cooperazione, vulnerabilità

Scheda attiva

  1. Riassumi la tesi del testo in modo preciso.
  2. Formula una possibile obiezione o una domanda critica.
